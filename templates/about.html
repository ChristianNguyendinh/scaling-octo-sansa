About

Web application that scrapes APIs to gather data about a person.

Right now gets data from Google's knowledge graph API, Facebook's Graph API, Imgur's API, Bing's (News) Search API, and Twitter's API.

It's called "Celeb Search" because if you try to search for a person without enough publicly available information from the various APIs, it crashes :P

Mainly this was just for me to practice django, react/js, rest APIs, and python
